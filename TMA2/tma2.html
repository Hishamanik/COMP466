How to setup the project:

  • Step 1: Install XAMPP and then open the XAMPP from the start menu.

  • Step 2: Start the Apache and MySQL Modules. Note: You might need to change your port 80 to 8080 and 443 to 4443 if you have skype installed in your system.

  • Step 3: Copy all the files to htdocs in the location where the XAMPP program is installed

  • Step 4: Go the phpmyadmin webiste by visiting http://localhost:8080/phpmyadmin and create new databases called task1_db and task2_db. Then copy the the files data of task1_db and task2_db inside part 1 and part 2 directory in their respective SQL box in phpmyadmin. This will create my MYSQL databases that will be used by the server.

  • Step 5: Now type in http://localhost:8080/index.html or the name of the folder you want to open and run (e.g part1/part2)
